Projects

News Summarizer

This Python-based desktop application combines Natural Language Processing (NLP) and sentiment analysis to extract, summarize, and evaluate news articles. Using Tkinter for the GUI, it provides a simple interface where users can enter a URL to fetch and analyze content. The app leverages Newspaper3k to extract article details like the title, author, and publication date, along with generating a concise summary. It also integrates TextBlob to assess the article's sentiment, giving insights into whether the content is positive, neutral, or negative. This tool makes it easy to quickly understand and evaluate news articles efficiently.

Credit Card Fraud Detection

This project uses Logistic Regression to detect fraudulent transactions from a Kaggle dataset. The dataset includes features like anonymized variables (V1-V28), transaction time, and amount, with a target variable indicating whether a transaction is legitimate or fraudulent. Due to the dataset's imbalance, under-sampling is applied to improve model performance. The data is preprocessed to handle missing values, and key features are analyzed for insights. The model is trained and tested to evaluate its accuracy in identifying fraudulent transactions effectively.
